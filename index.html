El Concepto de Bucle For

¿Qué es el Bucle For?

En programación, un bucle for es una estructura de control que permite ejecutar repetidamente un bloque de código un número específico de veces. Es especialmente útil cuando sabemos de antemano cuántas iteraciones necesitamos realizar.

Características Clave:

  • Controlado por contador: Utiliza una variable que se incrementa o decrementa en cada iteración.
  • Estructura definida: Tiene una inicialización, una condición de continuación y una expresión de incremento.
  • Eficiente para recorridos: Ideal para trabajar con arrays y colecciones de elementos.
  • Predecible: El número de iteraciones es conocido antes de entrar al bucle.

Sintaxis General:

// Sintaxis en la mayoría de lenguajes
for (inicialización; condición; incremento) {
    // Código a ejecutar en cada iteración
}

// Ejemplo específico
for (int i = 0; i < 5; i++) { // puede ser ++ que se incremente o -- que se decremente
    System.out.println("Iteración: " + i);
}

código de ejemplo con bucle for en el espacio:

El Concepto de Bucle While

¿Qué es el Bucle While?

El bucle while es una estructura de control que ejecuta repetidamente un bloque de código mientras una condición específica sea verdadera. A diferencia del bucle for, no necesitamos saber de antemano cuántas iteraciones se realizarán.

Características Clave:

  • Basado en condición: La ejecución continúa mientras la condición sea verdadera.
  • Evaluación al inicio: La condición se verifica antes de cada iteración.
  • Flexible: Puede ejecutarse cero o más veces dependiendo de la condición inicial.
  • Riesgo de bucle infinito: Si la condición nunca se vuelve falsa, el bucle se ejecutará indefinidamente.

Sintaxis General:

// Sintaxis básica
while (condición) {
    // Código a ejecutar
    // Es importante modificar algo que afecte la condición
}

// Ejemplo específico
int contador = 0;
while (contador < 5) {
    System.out.println("Contador: " + contador);
    contador++;
}

código de ejemplo con bucle while en el siguiente espacio:

Private Sub Button1_Click(sender As Object, e As EventArgs) Handles Button1.Click

El Concepto de Estructura Switch

¿Qué es la Estructura Switch?

La estructura switch es una sentencia de control que permite ejecutar diferentes bloques de código basándose en el valor de una expresión. Es una alternativa más legible y eficiente a usar múltiples sentencias if-else cuando tenemos que comparar una variable con múltiples valores posibles.

Características Clave:

  • Selección múltiple: Permite elegir entre varias opciones basadas en un valor.
  • Más legible: Mejora la claridad del código comparado con if-else anidados.
  • Caso por defecto: Puede incluir un caso default para manejar valores no especificados.
  • Sentencia break: Generalmente se usa break para evitar que se ejecuten casos siguientes.

Sintaxis General:

// FUNCIÓN CON SWITCH
function calcular(operacion) {
    // Obtener los valores
    let num1 = parseFloat(document.getElementById("num1").value);
    let num2 = parseFloat(document.getElementById("num2").value);
    let resultado;

    // Validar que sean números
    if (isNaN(num1) || isNaN(num2)) {
        mostrarResultado(" Por favor ingresa ambos números");
        return;
    }

    // ESTRUCTURA SWITCH
    switch (operacion) {
        case "+":
            resultado = num1 + num2;
            mostrarResultado(`${num1} + ${num2} = ${resultado}`);
            break;

        case "-":
            resultado = num1 - num2;
            mostrarResultado(`${num1} - ${num2} = ${resultado}`);
            break;

        case "*":
            resultado = num1 * num2;
            mostrarResultado(`${num1} × ${num2} = ${resultado}`);
            break;

        case "/":
            if (num2 === 0) {
                mostrarResultado(" No se puede dividir entre cero");
            } else {
                resultado = num1 / num2;
                mostrarResultado(`${num1} ÷ ${num2} = ${resultado.toFixed(2)}`);
            }
            break;

        default:
            mostrarResultado(" Operación no válida");
    }
}

function mostrarResultado(texto) {
    let divResultado = document.getElementById("resultado");
    divResultado.textContent = texto;
    divResultado.classList.add("mostrar");
}
