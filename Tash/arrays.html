Tutorial Interactivo sobre Arreglos

Aprende sobre arreglos unidimensionales, bidimensionales y multidimensionales con ejemplos prácticos y ejecutables.

Arreglos Unidimensionales

Un arreglo unidimensional es una colección de elementos del mismo tipo almacenados en una sola dimensión.

Ejemplo 1: Creación y Acceso
// Crear un arreglo de números let numeros = [10, 20, 30, 40, 50]; // Acceder a elementos por índice console.log(numeros[0]); // 10 console.log(numeros[2]); // 30 // Modificar un elemento numeros[1] = 25; console.log(numeros); // [10, 25, 30, 40, 50]
Ejemplo 2: Métodos Comunes
// Métodos de arreglos unidimensionales let frutas = ['manzana', 'banana', 'naranja']; // Agregar elemento al final frutas.push('uva'); console.log(frutas); // Eliminar último elemento frutas.pop(); console.log(frutas); // Obtener longitud del arreglo console.log(frutas.length); // 3 // Recorrer arreglo for (let i = 0; i < frutas.length; i++) { console.log(frutas[i]); }

Arreglos Bidimensionales

Un arreglo bidimensional es una estructura de datos que organiza elementos en filas y columnas, formando una tabla.

Ejemplo 1: Creación y Acceso
// Crear un arreglo bidimensional (matriz) let matriz = [ [1, 2, 3], [4, 5, 6], [7, 8, 9] ]; // Acceder a elementos console.log(matriz[0][0]); // 1 (fila 0, columna 0) console.log(matriz[1][2]); // 6 (fila 1, columna 2) // Modificar un elemento matriz[2][1] = 10; console.log(matriz[2]); // [7, 10, 9]
Ejemplo 2: Recorrido de Matriz
// Recorrer una matriz bidimensional let tabla = [ ['A', 'B', 'C'], ['D', 'E', 'F'], ['G', 'H', 'I'] ]; // Recorrer filas y columnas for (let fila = 0; fila < tabla.length; fila++) { for (let col = 0; col < tabla[fila].length; col++) { console.log(`Elemento en [${fila}][${col}]: ${tabla[fila][col]}`); } }

Arreglos Multidimensionales

Los arreglos multidimensionales son estructuras con más de dos dimensiones, útiles para representar datos complejos.

Ejemplo 1: Arreglo Tridimensional
// Crear un arreglo tridimensional let cubo = [ [ [1, 2], [3, 4] ], [ [5, 6], [7, 8] ] ]; // Acceder a elementos console.log(cubo[0][0][0]); // 1 console.log(cubo[1][1][1]); // 8 // Modificar un elemento cubo[0][1][0] = 9; console.log(cubo[0][1]); // [9, 4]
Ejemplo 2: Datos Estructurados
// Estructura de datos compleja con arreglos multidimensionales let estudiantes = [ [ ['Juan', 18, 'Matemáticas'], ['María', 17, 'Historia'] ], [ ['Pedro', 19, 'Física'], ['Ana', 16, 'Química'] ] ]; // Acceder a datos específicos console.log(estudiantes[0][0][0]); // 'Juan' console.log(estudiantes[1][1][2]); // 'Química' // Mostrar todos los estudiantes for (let grupo = 0; grupo < estudiantes.length; grupo++) { for (let estudiante = 0; estudiante < estudiantes[grupo].length; estudiante++) { console.log(estudiantes[grupo][estudiante].join(', ')); } }

Explicaciones Paso a Paso

Explicación: Arreglos Unidimensionales
Paso 1: Creación

Los arreglos unidimensionales se crean usando corchetes []. Cada elemento se separa por comas.

let miArreglo = [elemento1, elemento2, elemento3];
Paso 2: Acceso a Elementos

Los elementos se acceden mediante su índice, comenzando desde 0.

let primerElemento = miArreglo[0];
Paso 3: Modificación

Los elementos pueden modificarse asignando un nuevo valor a su posición.

miArreglo[1] = nuevoValor;
Paso 4: Métodos Útiles

JavaScript proporciona métodos como push(), pop(), length, etc., para manipular arreglos.

Explicación: Arreglos Bidimensionales
Paso 1: Creación

Los arreglos bidimensionales son arreglos que contienen otros arreglos.

let matriz = [[fila1], [fila2], [fila3]];
Paso 2: Acceso a Elementos

Se necesitan dos índices: el primero para la fila y el segundo para la columna.

let elemento = matriz[fila][columna];
Paso 3: Recorrido

Se utilizan bucles anidados: uno para las filas y otro para las columnas.

for (let i = 0; i < matriz.length; i++) { for (let j = 0; j < matriz[i].length; j++) { // Acceder a matriz[i][j] } }
Explicación: Arreglos Multidimensionales
Paso 1: Concepto

Los arreglos multidimensionales extienden el concepto a más de dos dimensiones, útil para datos complejos.

Paso 2: Creación

Se crean anidando arreglos dentro de otros arreglos.

let cubo = [[[capa1]], [[capa2]]];
Paso 3: Acceso

Se necesitan tantos índices como dimensiones tenga el arreglo.

let valor = cubo[x][y][z];
Paso 4: Aplicaciones

Se utilizan en gráficos 3D, análisis de datos complejos, simulaciones, etc.
