Introducción a los Arreglos

¡Bienvenido al curso interactivo de arreglos! Aprenderás de forma fácil y práctica cómo funcionan los arreglos en programación.

¿Qué es un arreglo?

Imagina que tienes una fila de cajas donde puedes guardar cosas. Cada caja tiene un número que la identifica (empezando desde 0). Eso es básicamente un arreglo.

En programación, un arreglo es una estructura que nos permite almacenar múltiples valores en una sola variable, organizados en orden.

Cada valor tiene una posición específica llamada índice, que comienza desde 0, no desde 1.

Arreglos Unidimensionales: Explicación

Paso 1: Creación

Los arreglos unidimensionales se crean usando corchetes []. Cada elemento se separa por comas.

let miArreglo = [elemento1, elemento2, elemento3];
Paso 2: Acceso a Elementos

Los elementos se acceden mediante su índice, comenzando desde 0.

let primerElemento = miArreglo[0];
Paso 3: Modificación

Los elementos pueden modificarse asignando un nuevo valor a su posición.

miArreglo[1] = nuevoValor;
Paso 4: Métodos Útiles

JavaScript proporciona métodos como push(), pop(), length, etc., para manipular arreglos.

// Añadir elemento al final miArreglo.push(nuevoElemento); // Eliminar último elemento miArreglo.pop(); // Obtener longitud let longitud = miArreglo.length;

Arreglos Bidimensionales: Explicación

Paso 1: Creación

Los arreglos bidimensionales son arreglos que contienen otros arreglos.

let matriz = [[fila1], [fila2], [fila3]];
Paso 2: Acceso a Elementos

Se necesitan dos índices: el primero para la fila y el segundo para la columna.

let elemento = matriz[fila][columna];
Paso 3: Recorrido

Se utilizan bucles anidados: uno para las filas y otro para las columnas.

for (let i = 0; i < matriz.length; i++) { for (let j=0; j < matriz[i].length; j++) { // Acceder a matriz[i][j] } }

Visualización: Un arreglo bidimensional es como una tabla con filas y columnas.

Para acceder a un valor, necesitas dos índices: matriz[fila][columna]

Por ejemplo, para acceder al valor 6 en la tabla, usarías matriz[1][1]

Práctica: Arreglos Bidimensionales

Ahora practica con arreglos bidimensionales. Modifica el código y observa cómo funciona.

// Crear una matriz bidimensional (tabla 2x3) let matriz = [ [8, 9, 7], [10, 6, 9] ]; // Acceder al elemento en la fila 1, columna 2 console.log("Elemento en [1][2]:", matriz[1][2]); // Modificar un elemento matriz[0][1] = 15; console.log("Después de modificar [0][1]:", matriz); // Recorrer toda la matriz console.log("Recorriendo la matriz:"); for (let fila = 0; fila < matriz.length; fila++) { for (let col=0; col < matriz[fila].length; col++) { console.log(`[${fila}][${col}]=${matriz[fila][col]}`); } }
Tip: En matrices bidimensionales, el primer índice siempre es la fila y el segundo la columna. Piensa en coordenadas (x,y) como en un plano cartesiano.

Arreglos Tridimensionales: Explicación

Paso 1: Concepto

Los arreglos multidimensionales extienden el concepto a más de dos dimensiones, útil para datos complejos.

Un arreglo tridimensional puede verse como un cubo de datos, donde necesitas tres coordenadas para acceder a un elemento específico.

Paso 2: Creación

Se crean anidando arreglos dentro de otros arreglos.

let cubo = [[[capa1]], [[capa2]]];
Paso 3: Acceso

Se necesitan tantos índices como dimensiones tenga el arreglo.

let valor = cubo[x][y][z];
Paso 4: Aplicaciones

Se utilizan en:

  • Gráficos 3D y videojuegos
  • Análisis de datos complejos
  • Simulaciones científicas
  • Imágenes médicas (como resonancias magnéticas)

Práctica: Arreglos Tridimensionales

Ahora practica con arreglos tridimensionales. Experimenta con el código para entender cómo funcionan las tres dimensiones.

Prueba este código:
// Crear un arreglo tridimensional (como un cubo) let cubo = [ [ // Capa 0 (Z=0) [1, 2], // Fila 0, Columnas 0 y 1 [3, 4] // Fila 1, Columnas 0 y 1 ], [ // Capa 1 (Z=1) [5, 6], // Fila 0, Columnas 0 y 1 [7, 8] // Fila 1, Columnas 0 y 1 ] ]; // Acceder a diferentes elementos console.log("Elemento en [0,0,0]:", cubo[0][0][0]); console.log("Elemento en [1,1,1]:", cubo[1][1][1]); // Modificar un elemento cubo[0][1][0] = 9; console.log("Después de modificar [0,1,0]:", cubo); // Recorrer todo el cubo console.log("Recorriendo el cubo:"); for (let z = 0; z < cubo.length; z++) { console.log(`Capa ${z}:`); for (let y=0; y < cubo[z].length; y++) { for (let x=0; x < cubo[z][y].length; x++) { console.log(` [${z},${y},${x}]=${cubo[z][y][x]}`); } } }
Tip: Los arreglos tridimensionales son útiles para representar datos en 3D, como coordenadas en el espacio (x,y,z) o datos organizados en capas.
